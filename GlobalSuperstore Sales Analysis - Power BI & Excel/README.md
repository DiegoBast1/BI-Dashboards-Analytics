# GlobalSuperstore Sales Analysis with Power BI and Excel

## Project Purpose
To analyze Total Sales by Year, Average Profit by Market, Average Profit by Category and Total Sales

## Data sources
- GlobalSuperstore csv file

## Tools Used
- Excel to store the data
- Power BI to load, clean and create visuals with data to answer this business questions

## Key Insights Found
- Technology is the most profitable category
- Unites States, APAC and the EU are the most profitable markets
- $2.19M in Total Sales from 2011 to 2015
- 2015 has a drop in Sales due to lack of information.
