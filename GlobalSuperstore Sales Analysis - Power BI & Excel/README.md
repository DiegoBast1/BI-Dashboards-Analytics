# GlobalSuperstore Sales Analysis with Power BI and Excel

## Project Purpose


## Preguntas de negocio respondidas
- ¿Cuál es la evolución mensual y anual de ventas?
- ¿Qué categorías y regiones generan mayor margen?
- ¿Qué días/horas tienen mayor ticket promedio?
- ¿Cuáles son los 10 productos con mayor rotación y rentabilidad?

## Fuente de datos
- Dataset público: [[nombre del dataset o link, ej: Kaggle Superstore Sales]]
- 150.000+ filas de transacciones

## Herramientas utilizadas
- SQL Server / PostgreSQL → Limpieza y transformación
- Power BI → Modelado de datos, DAX y visualizaciones interactivas
- Excel (opcional para validación)

## Principales insights encontrados
- Las ventas crecieron un **28 %** en promedio los sábados vs días laborables
- La categoría **Electrónicos** representa el 58 % del margen bruto
- 3 regiones concentran el **82 %** de las ventas totales
- Ticket promedio más alto: viernes 18-21 h (+34 %)
